# Echo Agenda - 心语计划

一个智能任务管理系统，专为缓解学生焦虑和提高效率而设计。

## 功能特性

### 🤖 AI 智能助手
- 支持 OpenRouter（免费路由）和 DeepSeek 两种 AI 服务
- 流式响应，实时显示 AI 回复
- 智能任务拆解和优先级建议
- 任务难度降级建议

### 🧩 可插拔任务管理
- **艾森豪威尔矩阵**：四象限任务分类
- **番茄钟**：25分钟专注工作法
- **吃青蛙**：优先完成最困难任务
- 所有插件可自由拖拽、最小化

### 🎨 用户体验
- 深色/浅色主题切换
- 流畅的动画效果
- 粒子特效反馈
- 完全响应式设计

### 💾 数据持久化
- LocalStorage 本地存储
- 自动保存所有状态
- 刷新页面完全恢复

## 使用方法

1. 在浏览器中打开 `echo-agenda.html`
2. 首次打开时会显示介绍弹窗，阅读功能说明
3. 勾选"不再显示此介绍"可跳过此弹窗
4. 点击右下角 ⚙️ 按钮配置 API 密钥
5. 选择 AI 服务并输入 API 密钥
6. 开始与 AI 对话，获取任务建议
7. 使用插件管理任务

## 获取 API 密钥

### OpenRouter（推荐，免费路由）
1. 访问 https://openrouter.ai/
2. 注册并登录账号
3. 在 Keys 页面创建 API 密钥
4. 选择 `openrouter/free` 模型路由

### DeepSeek
1. 访问 https://platform.deepseek.com/
2. 注册或登录账号
3. 进入 "API Keys" 页面
4. 创建 API 密钥

## 插件说明

### 🎯 艾森豪威尔矩阵
- 🔥 重要且紧急 - 立即做
- 📅 重要不紧急 - 计划做
- ⚡ 紧急不重要 - 授权做
- 🗑️ 不重要不紧急 - 断舍离

### 🍅 番茄钟
- 25分钟专注 + 5分钟休息
- 自动切换工作/休息模式
- 支持暂停和重置

### 🐸 吃青蛙
- AI 自动识别最困难任务
- 一键完成最重要的事
- 专注优先级管理

## 快捷操作

- **拖拽插件**：拖动标题栏移动插件位置
- **最小化插件**：点击"−"按钮收起插件
- **重置布局**：点击↻按钮恢复默认布局
- **主题切换**：点击🌙/☀️按钮切换主题
- **重新显示介绍**：在浏览器控制台输入 `showIntroAgain()`
