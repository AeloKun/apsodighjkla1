配置AI服务

可选择免费可用的 AI 提供商并配置 API 密钥。
推荐：OpenRouter Free Router（`openrouter/free`）

如何获取 OpenRouter API 密钥：

  1. 访问 https://openrouter.ai/
  2. 注册并进入 Keys 页面
  3. 创建 API Key（通常以 sk-or- 开头）
  4. 模型可使用 openrouter/free（免费路由）
  5. 复制API密钥到下方输入框

✅ 推荐：OpenRouter Free Router，适合免费试用
